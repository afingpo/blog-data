## 自用博客数据

若您来自[`友链 - 涵哲子居`](https://afipo.top/links)，希望与本站交换友链，则

### 注意事项：

1. 请使用在友链页所展示的`json`格式，自觉遵守`json`语法，若发现错误可帮助改正
2. 推荐您将本站友链也同样添加到您的友链页面

### 添加方法：

#### 通过 Pull Request
- 请在`./data/links.json`以`json`格式添加您的站点信息（可以第一条为参考）
- 请在添加信息成功后，发送一个`PR请求`(拉取请求)
- 您的站点数据将在审核后合并同时更新站点
- 若许久未通过审核/合并6小时后站点未更新，请发送邮件到 [邮箱](mailto:me@afipo.top)（主要周末/休息日回复，工作日小概率回复，请耐心等待）

格式：
```json
{
    "title": "",
    "imgurl": "",
    "desc": "",
    "siteurl": ""
}
```

#### 通过 Issue
- 请在 Issue 页使用`友链申请`模板并按要求填写相关信息
- 填写完成后，提交该 Issue
- 我们将会以 Gitbub Action 自动审理并提出PR + 人工合并拉取请求 的方式进行流程，并在通过后更新站点
- 若许久未通过审核/合并6小时后站点未更新，请发送邮件到 [邮箱](mailto:me@afipo.top)（主要周末/休息日回复，工作日小概率回复，请耐心等待）
