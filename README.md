## 自用博客数据

若您来自[`友链 - 涵哲子居`](https://blog.afiingpo.top/links)，希望与本站交换友链，则

有以下注意事项：

1. 请使用在友链页所展示的`json`格式，自觉遵守`json`语法，若发现错误可帮助改正
2. 推荐您将本站友链也同样添加到您的友链页面

有以下添加方法：
- 请在`./data/links.json`以`json`格式添加您的站点信息（可以第一条为参考）
- 请在添加信息成功后，发送一个`PR请求`(拉取请求)
- 您的站点数据将在审核后合并同时更新站点
- 若许久未通过审核/合并6小时后站点未更新，请发送邮件到`afingpo@465332.xyz`（主要周末/休息日回复，工作日小概率回复，请耐心等待）

格式：
```json
{
    "title": "",
    "imgurl": "",
    "desc": "",
    "siteurl": ""
}
```
